# 生産管理3級プランニング試験範囲カバー率分析レポート

## 概要

このレポートは、生産管理3級プランニング試験範囲と電気電子・機械設計向け講座の対応状況を分析したものです。

## ファイル構成

- `index.html` - メインレポート（HTML形式、GitHub Pagesで表示）
- `生産管理3級_カバー率分析レポート.md` - Markdown形式のレポート
- `生産管理3級_カバー率分析_詳細データ.csv` - 詳細データ
- `生産管理プランニング.csv` - 試験範囲データ
- `生産管理オペレーション.csv` - オペレーション試験範囲データ
- `電気電子・機械設計向け.csv` - 講座データ

## 機能

- 📊 インタラクティブなグラフとチャート
